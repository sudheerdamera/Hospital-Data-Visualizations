# 🏥 Hospital Data Analysis Dashboard

## 📌 Project Overview

This project is a **Power BI Hospital Data Dashboard** designed to analyze and visualize hospital-related data in an interactive and meaningful way. The dashboard helps in monitoring patient records, department performance, treatment insights, and overall hospital operations.

The goal of this project is to transform raw hospital data into actionable insights using data visualization techniques.

---

## 🚀 Features

* 📊 Interactive Power BI dashboard
* 🏥 Department-wise patient analysis
* 👨‍⚕️ Patient distribution visualization
* 📈 Trend analysis using line and combo charts
* 🥧 Pie chart representation for quick insights
* 📉 Performance monitoring and reporting
* 🔍 Easy-to-understand healthcare analytics
